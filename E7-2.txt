「3.差集合で関係除算を表現する」(P.129)で、除算を減算に還元して解く方法を紹介しました。
そのクエリを「厳密な除算」をするように修正してください(「厳密な除算」の定義、覚えていますか？)。
今度は過不足なくスキルを満たす社員だけを選択するので、結果は神埼氏一人になります。

(skills)             (EmpSkills)
+--------+          +-----------+--------+
| skill  |          | emp       | skill  |
+--------+          +-----------+--------+
| Java   |          | 平井      | C++    |
| Oracle |          | 平井      | Oracle |
| UNIX   |          | 平井      | Perl   |
+--------+          | 平井      | PHP    |
                    | 平井      | UNIX   |
                    | 渡来      | Oracle |
                    | 相田      | C#     |
                    | 相田      | Java   |
                    | 相田      | Oracle |
                    | 相田      | UNIX   |
                    | 神崎      | Java   |
                    | 神崎      | Oracle |
                    | 神崎      | UNIX   |
                    | 若田部    | Perl    |
                    +-----------+--------|

解答：
Java,Oracle,UNIXのスキル'のみ'持っている人だけ抽出します

SELECT                            -- 特定のスキルセットの社員を特定するため以下カラムを設定する
    emp                           -- 社員名カラムを設定する
FROM                              -- 情報取得のため以下テーブルを指定する
    EmpSkills ES1                 -- 社員スキルテーブルを指定する
WHERE                             -- スキル条件を指定する
    NOT EXISTS (                  -- 欲しいスキル以外のデータを除外する
        SELECT                    -- 必要なスキルセットを指定する
            skill                 -- スキルカラムを設定する
        FROM                      -- 必要なスキルセットを取得するために以下テーブルを指定する
            Skills                -- スキルテーブルを指定する
        EXCEPT                    -- 上記スキル以外のデータを特定する
        SELECT                    -- 必要なスキル以外のデータを特定するために以下カラムを設定する
            skill                 -- スキルカラムを設定する
        FROM                      -- スキル情報を以下テーブルより抽出する
            EmpSkills ES2         -- 社員スキルテーブルを指定する
        WHERE                     -- 情報を以下条件で関連づける
            ES1.emp = ES2.emp     -- 社員名の一致で関連づける
    )
GROUP BY                          -- スキル数を数えるために以下フィールドでグルーピングする
    emp                           -- 社員名でグルーピングする
HAVING                            -- スキルセット数が条件と一致するデータに絞り込む
    COUNT(skill) = (              -- skillについてカウントし
        SELECT                    -- 以下条件との一致で絞り込む
            COUNT(*)              -- スキル数カラムを設定する
        FROM                      -- 条件となるスキル数を取得するために以下テーブルを指定する
            Skills                -- スキルテーブルを指定する
    )
;

 emp
------
 神崎
